Step One · Wallet

Install MetaMask

~ 3 minutes

MetaMask is a free browser extension that holds your test wallet — a digital wallet where your test ETH, test USDC, and your tree NFTs will live. Think of it as a self-custodial bank account that nobody else can access.

  1. Open Chrome, Brave, or Firefox on a desktop computer (mobile works too, but desktop is easier for first-time setup).
  2. Go to the official MetaMask website using the button below — never download from an app store result, only from metamask.io.
  3. Click "Download" → "Install MetaMask for Chrome" (or whichever browser you use).
  4. Open the extension. Click "Create a new wallet" — you'll set a password and write down a 12-word backup phrase.
  5. Write that 12-word phrase down on paper and store it somewhere safe. This is your only key. We'll never see it. We can never recover it for you.
Install MetaMask →
Critical — read this

Anyone with your 12-word phrase has full control of your wallet. Never share it, never type it into a website, never store it in cloud notes. Even though this is a test wallet for now, building good security habits early matters.

Step Two · Network

Add Base Sepolia to MetaMask

~ 2 minutes

By default, MetaMask only knows about Ethereum mainnet. We need to tell it about Base Sepolia — Coinbase's free test version of the Base blockchain, where this platform lives during the preview.

The easiest way: one-click add. Use Chainlist, the standard tool for adding networks to MetaMask. It detects MetaMask automatically and adds the network with one click.

Add via Chainlist →

On Chainlist: search "Base Sepolia," click "Connect Wallet," then click "Add to MetaMask." MetaMask pops up to confirm. Click Approve.

Or add manually. If Chainlist isn't loading, open MetaMask → click the network dropdown at the top → "Add a network" → "Add a network manually" — then paste these values:

Don't confuse the two

Base Sepolia (chain ID 84532) is what you want. It's not the same as plain "Sepolia" (chain ID 11155111) — that's Ethereum's testnet, a different network entirely. If you accidentally add the wrong one, nothing breaks; just make sure you're on Base Sepolia before continuing.

Step Four · Currency

Get free test USDC

~ 1 minute

USDC is a stablecoin — a digital dollar that's always worth $1. The platform prices each tree at $5 USDC. On testnet, USDC is also free.

Use Circle's official faucet. Circle issues USDC, so their faucet is the most reliable source.

Open Circle USDC Faucet →

On the faucet:

  1. Connect your MetaMask wallet.
  2. Select "Base Sepolia" as the network.
  3. Select "USDC" as the asset.
  4. Request 10 USDC. Wait ~30 seconds.

After it arrives, you might not see USDC in MetaMask yet — that's because tokens need to be added manually. Open MetaMask → Tokens tab → "Import tokens" → paste:

Step Five · Plant

Plant your first tree

~ 2 minutes

Everything's set up. Now for the fun part — visit the live preview and complete a real on-chain donation.

Open the Mini App →

What to expect, step by step:

  1. Click "Connect Wallet". MetaMask pops up — approve the connection.
  2. The app verifies you're on Base Sepolia. If you're on the wrong network, it'll prompt you to switch — just click Approve.
  3. Choose how many trees you want to plant: 1 tree ($5), 5 trees ($25), or 10 trees ($50). Start with 1 to test.
  4. Click the green "Plant 1 Tree — $5" button.
  5. MetaMask popup #1 — Approve USDC. This gives the donation contract permission to pull $5 from your wallet. Click Confirm.
  6. MetaMask popup #2 — Donate. This is the actual donation + NFT mint. Click Confirm.
  7. Wait ~5 seconds. The success screen appears with your transaction hash.
  8. Click "View your dashboard →" to see your tree NFT.
What you should see in your wallet

After the transaction confirms: USDC balance drops by $5 (10 → 5), a new "Planet-A-mor Tree" NFT appears in MetaMask's NFT tab, and the dashboard shows your tree counted toward your impact.

Plant a few more if you want — the app handles single donations and bulk donations the same way. Each one mints a new NFT certificate.

FAQ

Common questions

Is this real money?
No. Everything you do here uses test tokens that are free, unlimited, and worth nothing outside the testnet. You cannot lose real money on Base Sepolia.
My MetaMask says "insufficient funds"
You're either on the wrong network (check the dropdown at the top of MetaMask says "Base Sepolia") or out of test ETH. Hit the Coinbase faucet again.
The first popup worked, the second one says "exceeds gas limit"
This usually self-resolves on retry — just click Plant a Tree again. The first popup grants USDC approval; the second is the actual donation. Sometimes the wallet needs a moment to register the approval before the donation can be simulated.
Where can I see my donation publicly?
Every donation creates an event on Base Sepolia, viewable forever at sepolia.basescan.org. Your transaction hash on the success screen links directly to it.
Is my NFT the same as a real ownership certificate?
During the testnet preview, NFTs are real on-chain ERC-721 tokens but represent test donations only — the trees aren't planted from your test transaction. On mainnet, every NFT will represent a verified, real tree.
I found a bug. How do I report it?
Email contact@planet-a-mor.org with a screenshot, the URL you were on, and a one-line description. Your feedback directly shapes the mainnet release.
